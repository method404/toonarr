# Toonarr v0.1.0

## 요약

첫 공개 버전입니다. 네이버 웹툰 개인 수집을 위한 기본 관리자 UI와 로컬 저장 흐름을 정리했습니다.

## 포함 내용

- Sonarr 스타일 라이브러리 화면 구성
- 작품 검색, 요일별, 완결무료 탐색 화면 추가
- 작품 상세, 회차 상태, 메타데이터 표시
- 시리즈 메타데이터 및 회차별 저장 구조 구현
- 백그라운드 저장 큐와 작업 이력 기본 흐름 반영
- 네이버 계정 저장 및 브라우저 로그인 흐름 추가
- 도네이트 페이지 및 업데이트 확인 화면 추가

## 주의사항

- 아직 개발 중인 초기 버전입니다.
- 네이버 로그인 및 성인 콘텐츠 접근은 외부 정책 변화에 영향을 받을 수 있습니다.
- 자동화 스케줄, 인증 복구, 배포 업데이트는 계속 보완 중입니다.

## 면책

- 이 프로젝트는 개인 저장용입니다.
- 수집 결과물의 공유, 재배포, 상업적 사용은 사용자 책임입니다.
- 계정 제한, 데이터 손실, 서비스 차단, 법적 문제 등 사용 중 발생하는 모든 문제는 사용자 본인 책임입니다.
